### 실행 가이드
1. 프로젝트를 clone 한다.
```shell script
git clone https://github.com/umanking/taxi-dispatch-app.git
```

2. 프로젝트를 open한다. 
3. 루트 경로에서 해당 명령어를 입력한다.
```shell script
$ mvn clean compile build
$ mvn spring-boot:run
```
